# An Online Quiz app 


## Features are: 
1. Different subjects
2. Different sets in each subject  
3. MCQ question
4. Timer and show Output

## Prerequisites
* Installation of Android Studio
* Android Studio Version 3.6

## Language & Technologies used:
* JAVA
* Firebase
